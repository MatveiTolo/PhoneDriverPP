# 📱 PhoneDriverPP

Управление Android телефоном через ADB из командной строки.

## 🚀 Быстрый старт

### 1. Установите ADB

Скачайте [Platform Tools] и добавьте в PATH.

### 2. Подготовьте телефон

- Включите **"Режим разработчика"** (нажмите 7 раз на "Номер сборки")
- Включите **"Отладку по USB"**
- Подключите телефон кабелем и нажмите **"Разрешить"**

### 3. Запустите проект

```bash
git clone https://github.com/ВАШ_АККАУНТ/PhoneDriverPP.git
cd PhoneDriverPP
pip install -r requirements.txt
python driver.py
